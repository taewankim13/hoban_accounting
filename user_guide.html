📅 월별 필터

상단 월 버튼을 클릭하면 해당 월 데이터만 필터링됩니다. '전체'로 전체 기간 조회가 가능합니다.

📤 CSV 데이터 업로드

ERP에서 추출한 CSV 파일을 드래그&드롭으로 업로드합니다. 업로드 즉시 18개 탐지 룰이 자동 분석됩니다.

🚨 위험등급 통계

High/Medium/Low/정상 4단계로 분류된 전표 수를 실시간으로 보여줍니다.

🏗️ 프로젝트별 현황

상위 15개 현장의 전표 현황과 고위험 건수를 표시합니다. '검토' 버튼으로 해당 현장 전표를 바로 확인합니다.

버튼위치기능 설명
월 버튼상단 월별 필터 영역선택한 월의 데이터만 표시합니다. 여러 월 데이터가 있을 때 사용합니다.
+ 데이터 업로드월별 필터 우측CSV 파일 업로드 모달을 엽니다. 월을 선택하고 파일을 첨부합니다.
검토현장별 테이블 행해당 현장의 전표 목록을 전표 검토 화면에서 필터링하여 보여줍니다.
전표 생성우측 하단 빠른 접근전표 생성 화면으로 이동합니다.
긴급 검토우측 하단 빠른 접근고위험(High) 전표만 필터링한 검토 화면으로 이동합니다.
X (삭제)월별 데이터 관리해당 월의 전체 데이터를 삭제합니다. 삭제 전 확인을 요청합니다.

📸 AI OCR 분석

영수증/세금계산서 이미지를 업로드하면 Gemini AI가 금액, 거래처, 날짜, 부가세를 자동 인식합니다.

💬 대화형 생성

"자재 500만원" 같은 자연어로 AI에게 전표 생성을 요청할 수 있습니다.

✅ 자동 검증

저장 시 차대변 일치, OCR 값 매칭, 계정과목 방향을 자동 검증합니다.

📎 연결문서

기안서, 계약서 등 관련 문서를 첨부할 수 있습니다. PDF/이미지는 AI가 자동 분석합니다.

버튼위치기능 설명
+ 파일 선택증빙자료 영역영수증/세금계산서 이미지를 업로드합니다. 업로드 즉시 AI OCR 분석이 시작됩니다.
전송AI 챗 입력창자연어 메시지를 AI에게 전송합니다. AI가 전표 양식을 자동 채워줍니다.
+ 항목 추가라인 항목 테이블차변/대변 라인 항목을 추가합니다.
초기화전표 양식 하단입력된 전표 양식 전체를 초기 상태로 되돌립니다.
전표 저장전표 양식 하단전표를 검증 후 저장합니다. OCR 불일치나 차대변 방향 의심 시 확인 팝업이 표시됩니다.

🔴 위험등급 필터

좌측 사이드바에서 위험등급별로 필터링합니다. 고위험 전표를 우선 검토할 수 있습니다.

🏷️ 이상사유 필터

E001~E018 이상사유 코드로 필터링합니다. 여러 코드를 동시 선택하면 AND 조건으로 필터됩니다.

🔍 다차원 검색

전표번호, 일자, 적요, 입력자, 현장, 거래처, 계정과목 등 다양한 필드로 검색할 수 있습니다.

🤖 AI 검토 제안

전표 상세 팝업에서 Gemini AI가 각 위반 항목별 점검 포인트, 수정 방법, 검토 절차를 안내합니다.

버튼위치기능 설명
위험등급 버튼좌측 사이드바High/Medium/Low/정상별 전표를 필터링합니다.
이상사유 코드좌측 사이드바특정 위반 유형으로 필터링합니다. 복수 선택 시 AND 조건입니다.
검색상단 필터 바입력한 검색어로 전표를 검색합니다. 필드를 선택하여 정밀 검색이 가능합니다.
차/대변 불일치 검토상단 필터 바차변/대변 방향이 의심스러운 전표를 순차적으로 검토하는 전용 모달을 엽니다.
OCR 값 일치 점검상단 필터 바영수증 OCR 분석값과 입력값을 비교 검토하는 전용 모달을 엽니다.
전표 행 클릭테이블 본문전표 상세 팝업이 열립니다. AI 분석, 증빙자료, 연결문서, AI 검토 제안을 확인합니다.
검토 승인상세 팝업 하단'제출' 상태의 전표를 '검토완료'로 승인합니다.
반려상세 팝업 하단전표를 '반려' 상태로 변경합니다.

🤖 AI 룰 생성기

자연어로 탐지 룰을 설명하면 AI가 조건을 파싱하여 자동 생성합니다.

🔧 조건 편집기

필드, 연산자, 값을 조합하여 세밀한 탐지 조건을 설정합니다. AND/OR 논리를 지원합니다.

🔄 전체 재분석

룰 변경 후 전체 전표를 재분석합니다. 진행 상황이 실시간으로 표시됩니다.

🎚️ 활성/비활성

각 룰의 토글 스위치로 즉시 활성화/비활성화할 수 있습니다.

버튼위치기능 설명
전송AI 룰 생성기자연어 설명을 AI에게 전송하여 탐지 룰 조건을 자동 생성합니다.
예시 버튼AI 룰 생성기 하단클릭하면 해당 예시가 채팅창에 자동 입력됩니다.
+ 조건 추가조건 편집기AND/OR 조건을 추가로 설정합니다.
룰 저장조건 편집기 하단룰을 저장합니다. 기존 전표에는 즉시 적용되지 않습니다.
저장 + 재분석조건 편집기 하단룰을 저장하고 즉시 전체 전표를 재분석합니다.
+ 수동 추가룰 목록 상단빈 편집 모달을 열어 수동으로 룰을 추가합니다.
전체 재분석룰 목록 상단현재 활성화된 모든 룰로 전체 전표를 재분석합니다.
수정룰 카드 우측해당 룰의 조건을 편집 모달에서 수정합니다.
삭제룰 카드 우측해당 룰을 삭제합니다. 삭제 전 확인을 요청합니다.
