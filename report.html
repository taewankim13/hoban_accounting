1.시스템 개요

본 시스템은 호반건설 ERP 전표 데이터를 대상으로, 사내 구축형 LLM(PwC Workspace Gemini 3.1 Pro)과 규칙 기반 엔진을 결합하여 이상 전표를 자동 탐지하고, 영수증·세금계산서 이미지를 AI로 분석하여 전표 생성부터 검토까지 전 과정을 자동화하는 지능형 감사 플랫폼입니다.

3.화면 구성 및 주요 기능

3.1 대시보드 (/)

전체 이상탐지 현황을 한눈에 파악하는 통합 현황 화면입니다.

  • 위험등급 요약 — 전체/고위험/주의/경고/정상 건수 및 비율을 카드형 지표로 표시
  • 프로젝트별 현황 — 각 프로젝트의 전표 건수, 총 금액, 위험 비율을 테이블로 제공
  • 전표 유형 분포 — 일반전표, 대체전표, 역분개전표 등 유형별 분포 표시
  • 월별 데이터 관리 — 월 선택 필터, CSV 데이터 업로드, 월별 데이터 삭제 기능

3.2 전표 생성 (/create)

AI 보조를 통한 신규 전표 생성 화면입니다.

  • 영수증/세금계산서 이미지 업로드 — LLM Vision OCR로 금액, 거래처, 일자, 부가세를 자동 추출하여 전표 폼에 반영
  • AI 챗 어시스턴트 — 자연어로 전표 생성/수정 요청 (예: "자재 구매 500만원", "금액을 1,155만원으로 수정해줘")
  • 계정과목 검색 — 1,418개 계정과목 실시간 검색, 차대변 방향 자동 검증
  • 거래처 검색 — 코드/이름/유형별 거래처 조회
  • 전표 라인 관리 — 차변/대변 항목 추가·삭제, 차대변 합계 자동 계산 및 균형 검증
  • 자동 분석 — 전표 저장 시 21개 룰 자동 적용, 위험등급 즉시 판정

3.3 전표 검토 (/review)

탐지된 이상 전표를 검토하고 승인/반려 처리하는 핵심 업무 화면입니다.

  • 좌측 사이드바 필터 — 위험등급(고위험/주의/경고) 및 이상사유 코드별 필터링
  • 카드뷰/테이블뷰 전환 — 전표 목록을 카드형 또는 테이블형으로 전환하여 조회
  • 전표 상세 모달 — 전표 헤더, 라인 항목, 증빙 파일, 연결문서, AI 분석 결과를 탭으로 구분하여 표시
  • 증빙 관리 — 증빙 파일 업로드·삭제, LLM 기반 계약서/공과금 고지서 자동 파싱
  • 연결문서 관리 — 기안서/품의서/검수확인서 업로드·삭제, LLM 자동 파싱
  • OCR 재분석 — 영수증 이미지 OCR 결과 재실행, 전표 금액과 비교 검증
  • 승인/반려 — 검토 완료 처리 및 반려(사유 입력) 기능
  • 검색/정렬 — 전표번호, 적요, 거래처명 키워드 검색, 금액순·위험도순 등 정렬
  • 페이지네이션 — 대량 데이터 처리를 위한 페이지 단위 조회 (기본 30건)

3.4 룰 관리 (/rules)

이상탐지 룰을 관리하고, 자연어로 새로운 룰을 생성할 수 있는 관리 화면입니다.

  • 룰 카드 목록 — 각 룰의 명칭, 조건, 심각도를 색상 코딩된 카드로 표시
  • 룰 활성화/비활성화 — 개별 룰을 토글하여 분석 적용 여부 제어
  • 룰 편집/삭제 — 기존 룰의 조건, 심각도, 설명 등을 수정
  • 자연어 룰 생성 — 챗 인터페이스에서 자연어(예: "금액이 5억 초과이면 고위험 경고")로 룰을 자동 생성
  • 수동 룰 추가 — 필드, 연산자, 값, 심각도를 직접 지정하여 룰 생성
  • 전체 재분석 — 현재 룰 세트로 모든 전표를 일괄 재분석, 진행률 표시

4.AI/LLM 기반 지능형 기능

모든 AI 기능은 PwC Workspace Gemini 3.1 Pro API 하나로 통합되어 있으며, API 키는 사용자 브라우저의 localStorage에 저장되어 서버에 보관하지 않는 보안 구조입니다.
📷

영수증/세금계산서 OCR

Gemini Vision API로 이미지에서 일자, 거래처, 사업자번호, 공급가액, 부가세, 합계금액, 품목을 추출합니다. 이미지 자동 리사이즈(1.5MB 이하, 2000px 이하), PDF 1페이지 자동 변환을 지원합니다.

💬

대화형 전표 어시스턴트

자연어로 전표를 생성/수정합니다. "자재 구매 500만원", "부가세 분리해줘", "금액을 1,155만원으로 수정" 등의 대화형 명령을 이해하고 계정과목, 차대변 라인을 자동 생성합니다. API 키 없이도 로컬 파서로 기본 기능을 제공합니다.

📜

증빙문서 자동 분석

계약서에서 계약기간, 계약금액, 계약대상 등을 추출하고, 공과금 고지서에서 당월요금, 기본월, 납기일을 추출합니다. 도급기성 문서의 금회기성액도 자동 인식합니다.

📚

자연어 룰 생성

"거래처가 개인이고 금액이 1천만원 이상이면 고위험"과 같은 자연어를 35+ 필드 타입의 구조화된 룰 JSON으로 자동 변환합니다. AND/OR 복합 조건도 지원합니다.

💰

계정과목 자동 매핑

OCR 추출 결과의 품목 키워드(철근, 주유, 택시, 렌탈 등)를 기반으로 적합한 차변/대변 계정과목을 자동 제안합니다. 1,418개 계정과목 마스터를 참조합니다.

🔒

API 키 관리

설정 모달에서 Gemini API Key를 입력하면 브라우저 localStorage에 안전하게 저장됩니다. window.fetch를 오버라이드하여 LLM 호출 시 자동으로 인증 헤더를 주입합니다. 키 미입력 시 팝업으로 즉시 입력을 요청합니다.

4.1 AI 처리 흐름

  1. 이미지 업로드 — 사용자가 영수증/세금계산서 이미지를 드래그앤드롭 또는 파일 선택으로 업로드
  2. 전처리 — Pillow로 이미지 리사이즈 (1.5MB/2000px 이하), PDF는 PyMuPDF로 첫 페이지 PNG 변환
  3. LLM Vision 호출 — Base64 인코딩 후 Gemini Vision API에 구조화된 프롬프트와 함께 전송
  4. JSON 파싱 — LLM 응답에서 금액, 거래처, 일자, 부가세 등 구조화 데이터 추출
  5. 계정 매핑 — 품목 키워드 기반으로 차변/대변 계정과목 자동 제안
  6. 폼 반영 — 추출된 데이터를 전표 생성 폼에 자동 입력, 사용자 확인 후 저장

5.2 룰 엔진 구조

  • 단일 조건 룰 — 하나의 필드-연산자-값 조합으로 평가 (예: max_line_amount > 1000000000)
  • 복합 조건 룰 — 다중 조건을 AND/OR 로직으로 결합 (예: C011은 3개 조건 AND)
  • 위험 점수 계산 — 각 룰의 가중치(score)를 합산, 0~1.0 범위로 정규화
  • 위험등급 자동 판정 — High(≥0.8) / Medium(≥0.3) / Low(<0.3) 자동 분류
  • 지원 필드 35+ — 금액, 일자, 계정, 거래처, 적요, 증빙 필드, 연결문서 필드 등 광범위한 조건 지원

7.개선된 특장점

🏠

사내 구축형 AI

PwC Workspace 내 Gemini API를 활용하여 외부 클라우드에 데이터가 유출되지 않는 폐쇄망 운영이 가능합니다. API 키는 서버에 저장하지 않고 사용자 브라우저에서 관리합니다.

⚡

통합 AI 키 관리

OCR, 챗 어시스턴트, 자연어 룰 생성, 증빙 분석까지 Gemini API Key 하나로 모든 AI 기능을 통합 사용합니다. window.fetch 오버라이드로 기존 코드 수정 없이 자동 헤더 주입합니다.

📊

건설업 특화 룰

도급기성 금회기성액, 임차료-월세액 일치, 교육훈련비 연결문서 누락 등 건설업 회계 실무에 특화된 21개 이상탐지 룰을 기본 제공합니다.

📄

증빙 연계 분석

전표 데이터만이 아닌 첨부 증빙(계약서, 고지서)과 연결문서(기안서, 품의서)까지 AI로 분석하여 전표-증빙 간 금액·기간 불일치를 자동 탐지합니다.

📝

자연어 룰 생성

감사인이 코딩 없이 자연어로 새로운 탐지 룰을 생성할 수 있어 현장 감사 상황에 즉시 대응 가능합니다. AND/OR 복합 조건도 자연어로 표현 가능합니다.

🔄

Graceful Degradation

API 키가 없어도 로컬 파서로 기본 기능을 제공합니다. 챗 어시스턴트의 금액 수정, 부가세 분리, 계정 변경, 10개 건설업 거래 패턴 파싱이 오프라인에서도 동작합니다.

📦

ERP 데이터 자동 임포트

호반건설 ERP CSV 파일을 v1/v2 형식 자동 감지하여 임포트합니다. 중복 컬럼 처리, 인덱스 기반 계정코드 매핑, 프로젝트 자동 생성으로 수작업을 최소화합니다.

🚀

원클릭 배포

Docker 컨테이너 + Railway를 통해 Git push만으로 자동 빌드 및 배포됩니다. 컨테이너 시작 시 DB 초기화, 데이터 임포트, 룰 분석이 자동 실행됩니다.
