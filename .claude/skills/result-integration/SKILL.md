# 결과 통합 및 웹 UI

FDS 시스템의 웹 애플리케이션과 API를 관리한다.

## 기술 스택
- **백엔드**: FastAPI (Python), SQLAlchemy, SQLite
- **프론트엔드**: Jinja2 템플릿, Vanilla JS, 호반건설 CI CSS
- **CI 색상**: Orange #EE7500, Gray #89898A, Dark #575553

## 화면 구성

### 대시보드 (/)
위험등급 통계 + 프로젝트별 현황 테이블 + 전표유형별 통계

### 전표 생성 (/create)
- 상단: 영수증 업로드(300px) + AI 챗(나머지)
- 하단: 전표 입력 폼 (헤더 6열 + 거래처유형/거래처/적요 중간행 + 차대변 테이블)
- 검색형 입력: 프로젝트/계정과목/거래처 드롭다운 (200ms 디바운스, 30건)

### 전표 검토 (/review)
- 필터: 위험등급(pill 버튼) + 이상사유(pill 버튼) + 프로젝트/전표유형/거래처유형(드롭다운)
- 보기: 카드 뷰(3열 그리드) / 테이블 뷰(mousedown 정렬, 드래그앤드롭 비활성)
- 상세 모달: 3열 헤더 정보 + 영수증 이미지 + 전표 항목(거래처유형 컬럼 포함) + AI 분석
- OCR 점검: 건별 비교 팝업 (이미지 | 비교 테이블) + AI 재분석
- 페이지네이션: 50건 단위

### 룰 관리 (/rules)
- AI 룰 빌더: 좌(대화) / 우(필드 편집) 분할
- 룰 목록: 2열 카드 그리드, 카테고리별 그룹, 검출 건수 뱃지
- 재분석: 백그라운드 스레드, 진행률 게이지바, 완료 체크, 룰 변경 알림 배너
